## Quick start

### Install
For PyQt5:
```shell
pip install PyQt5-Frameless-Window
```
For PyQt6:
```shell
pip install PyQt6-Frameless-Window
```
For PySide2:
```shell
pip install PySide2-Frameless-Window
```
For PySide6:
```shell
pip install PySideSix-Frameless-Window
```


### Requirements

| Platform | Requirement |
| :------: | :---------: |
|  Win32   |   pywin32   |
|  Linux   |   xcffib    |
|  MacOS   |   pyobjc    |
